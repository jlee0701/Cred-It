Credit Card Database:

This program provides graphic user interface that gives four options to: search, insert, update, and delete an item. 
The main functionalities are
- Make A Selection
- Results

To make a selection: Choose among the six entities that will list all attributes. Full statements must be entered as well as the corresponding checkbox 

Results: Results will vary depending on what is checked 


The 6 entities 
-Member
-CardReward
-HasReward
-Issuer
-Member
-Own
-PurchasingPattern

Select queries:
1. Select cards that offer selected rewards
2. Select member owned cards

Insert:
1. Add new card to members

Update:
1. Update member income
2. Update member email

Delete:
1. Delete a member owned card by ssn and card number
